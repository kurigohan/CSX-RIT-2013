Lab - Activity 2

In this activity write a program named, ScramblePad.java, that will create a GUI similar to the one displayed below:

Be sure to pay close attention to the colors used in the GUI.  Your program must create a GUI that uses the same colors and the same layout used in the image above. 

As in the previous activity your GUI does not have to be active (i.e. the buttons contained in the GUI do not need to work).  You simply have to write a program that creates a GUI similar to the one shown above.  Your program does not need to terminate correctly, you can use the Del or the control-c key to terminate the program.

Once you are convinced that your program is working correctly, submit it using the following command:

try csx-grd lab13-2 ScramblePad.java
